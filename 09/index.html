位取り記法

以下の仕様を満たすように、関数を定義してください。定義は 09.js に記述してください。

利用例


countDigits(1) == 1;    // true
countDigits(12) == 2;   // true
countDigits(123) == 3;  // true
countDigits(1234) == 4; // true

countDigits(-1) == -1;  // true
countDigits(-12) == -1; // true

countDigits(31415.9265) == 5; // true

仕様

  • 関数名：countDigits
  • 引数：1 つの数
  • 返り値：引数に指定された数値の桁数。ただし負の数が指定された場合は - 1 を返す。また小数点以下は切り捨てて計算する。
