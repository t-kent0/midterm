FizzBuzz 問題

以下の仕様を満たすように、関数を定義してください。定義は 06.js に記述してください。

利用例


fizzbuzz(1) == 1; // true
fizzbuzz(2) == 2; // true
fizzbuzz(3) == "fizz"; // true
fizzbuzz(4) == 4; // true
fizzbuzz(5) == "buzz"; // true
fizzbuzz(6) == "fizz"; // true
fizzbuzz(7) == 7; // true
fizzbuzz(8) == 8; // true
fizzbuzz(9) == "fizz"; // true
fizzbuzz(10) == "buzz"; // true
fizzbuzz(11) == 11; // true
fizzbuzz(12) == 12; // true
fizzbuzz(13) == 13; // true
fizzbuzz(14) == 14; // true
fizzbuzz(15) == "fizzbuzz"; // true

仕様

  • 関数名：fizzbuzz
  • 引数：1 つの整数値（1以上）
  • 返り値：
    • 引数が 3 の倍数の場合：fizz という文字列を返す
    • 引数が 5 の倍数の場合：buzz という文字列を返す
    • 引数が 3 の倍数で、しかも 5 の倍数の場合：fizzbuzz という文字列を返す
    • それ以外：引数の数値を返す
