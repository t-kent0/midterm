文字列の操作

以下の仕様を満たすように、文字列操作を行う関数を 2 つ定義してください。定義は 05.js に記述してください。

利用例


var text_a = "Hello, ";
var text_b = "world!";

var length_a = strlen(text_a);
length_a == 7; // true

var length_b = strlen(text_b);
length_b == 6; // true

var new_text = concat(text_a, text_b); // "Hello, world!"
var length = strlen(new_text);
length == 13;

仕様

  • 関数名：strlen
    • 引数：1 つの文字列
    • 返り値：引数に指定された文字列の長さを返す
  • 関数名：concat
    • 引数：2 つの文字列
    • 返り値：第一引数の文字列の後ろに、第二引数の文字列を結合した文字列を返す
