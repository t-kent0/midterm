距離関数

以下の仕様を満たすように、関数を定義してください。定義は 07.js に記述してください。

利用例


var zero = {x: 0, y: 0};
var a = {x: 1, y: 0};
var b = {x: 1, y: 1};
var c = {x: 2, y: 7};
distance(a, a) == 0;
distance(zero, a) == 1;
distance(zero, b) == 2;
distance(b, c) == 7;

仕様

  • 関数名：distance
  • 引数：2 つの二次元平面上の点を表すオブジェクト（後述）
  • 返り値：引数に指定された 2 つの点のマンハッタン距離

2 次元平面上の点を表すオブジェクトは次の 2 属性を持つ

  • x: X 座標の値
  • y: Y 座標の値
