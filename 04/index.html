論理演算

以下の仕様を満たすように、論理演算を行う関数 xor を定義してください。定義は 04.js に記述してください。

利用例


xor(true, true);   // false
xor(false, true);  // true
xor(true, false);  // true
xor(false, false); // false

仕様

  • 関数名：xor
  • 引数：2つの真理値
  • 返り値：以下の表にしたがうとする
第1引数第2引数返り値
truetruefalse
truefalsetrue
falsetruetrue
falsefalsefalse
