足し算と引き算を行う計算機

以下の仕様を満たすように、関数を定義してください。定義は 10.js に記述してください。

利用例


var exp1 = [1, "+", 1];
var exp2 = [1, "-", 1];
var exp3 = [1, "+", 1, "-", 2];
var exp3 = [1, "+", 1, "-", 2, "+", -10];

calc(exp1) == 2; // true
calc(exp2) == 0; // true
calc(exp3) == 0; // true
calc(exp4) == -10; // true

仕様

  • 関数名：calc
  • 引数：1 つの配列
  • 返り値：配列で表された式を評価して、その計算結果を返す：
    • 上記の exp1 のように、[数値, "+" , 数値] の場合、二つの数値の足し算を行う
    • 上記の exp2 のように、[数値, "-" , 数値] の場合、二つの数値の引き算を行う
    • "+", "-" のことを演算子と呼び、演算子はこの 2 種類しか存在しない
    • 上記の exp3 のように、[数値1, 演算子1 , 数値2, 演算子2, 数値3 …] となっている場合、最初の 3 つの演算を行い、 [数値1, 演算子1 , 数値2] の計算を行い、その結果を a を使って [a, 演算子2, 数値3] の計算を行う。これを配列の要素がなくなるまで繰り返す
