恒等写像

引数に指定された値を、そのまま返す関数 id を 01.js に定義してください。

利用例


var result = id(1);
result == 1; // true;

var text = id("hello");
text == "hello"; // true

仕様

  • 関数名：id
  • 引数の数：1
