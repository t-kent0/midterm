四則演算

以下の仕様を満たすように、5 つの四則演算を行う関数を定義してください。定義は 02.js 記述してください。

利用例


var one = plus(1, 0);
one == 1; // true;

var zero = minus(one, 1);
zero == 0; // true;

var eight = times(2, 4);
eight == 8; // true

var two = div(eight, 4);
two == 2; // true

var rest = mod(eight, 7);
rest == one; // true

仕様

関数名引数の数返り値
plus2第1引数に第2引数を足した結果
minus2第1引数から第2引数を引いた結果
times2第1引数に第2引数をかけた結果
div2第1引数を第2引数で割った結果
mod2第1引数を第2引数で割った余り

引数は全て数値が与えられるとする。
